# 📚 Discord.js Collection Rehberi

Bu depo, Discord botları kodlarken en çok kullanılan veri yapılarından biri olan **Discord.js Collection** sınıfının detaylı kullanım rehberini ve örnek kodlarını barındırmaktadır. (Bu rehber Webailesi platformu için özel olarak hazırlanmıştır.)

## 🚀 Neler Öğreneceğiz?
Bu repo içerisindeki `ornekler.js` dosyasında, Collection'ın en çok kullanılan şu metotlarının basit ve anlaşılır örnekleri bulunmaktadır:
- `set()` ve `get()` (Sisteme veri ekleme ve çekme)
- `has()` (Bir verinin sistemde olup olmadığını kontrol etme)
- `find()` (Belirli bir şarta uyan ilk veriyi anında bulma)
- `filter()` (İstediğimiz şarta uyan tüm verileri toplu şekilde filtreleme)
- `map()` (Verilerin içinden sadece istediğimiz kısımları alıp liste yapma)
- `sort()` (Kişileri, levelleri veya puanları belirli bir düzene göre sıralama)

## 💻 Nasıl Çalıştırılır?
Eğer `ornekler.js` dosyasını kendi bilgisayarınızda canlı olarak test etmek ve çıktıları görmek isterseniz şu adımları izleyebilirsiniz:

1. Bilgisayarınızda **Node.js**'in kurulu olduğundan emin olun.
2. Bu projeyi bilgisayarınıza indirin veya klasöre çıkartın.
3. Proje klasörünün içinde boş bir alana sağ tıklayıp terminalinizi (CMD / PowerShell) açın.
4. Gerekli modülü indirmek için şu komutu girin:
   ```bash
   npm install discord.js
   ```
5. Sonrasında örnek kodları çalıştırmak için şu komutu girin:
   ```bash
   node ornekler.js
   ```

Tüm konsol çıktıları terminalinizde adım adım görünecektir! İyi kodlamalar dilerim.
